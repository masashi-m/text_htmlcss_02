◆id 属性と class 属性

主に HTML タグと CSS プロパティを連携させる際に設定します。
これまでに学習した「a タグの href 属性」や 「img タグの src 属性」と異なり、
タグの種類に関わらず属性を指定します。



●id 属性

id 属性は、ページ内で固有の名前を付けるための属性(識別子)です。
固有の属性のため、同ページ内で他のタグに対し同一の id 値を指定したり、
1 つの要素に複数の id 属性を付ける事ができません。
HTML での記述方法は以下の通りです。

[html]

<p id="first">テキスト1</p>
<p id="second">テキスト2</p>

CSS で id 属性を指定する際は、#を付けて id 名を記述します。
また、セレクタ(タグ名)の記述は省略可能です。

[CSS]

#first {
  color: red;
}

●class 属性

id 属性と違い、class 属性は同ページ内で何度も使う事ができます。
また、半角スペースで区切る事で、複数の class を付与する事も可能です。

[html]

<p class="first">テキスト1</p>
<p class="first second">テキスト2</p>
<p class="first second third">テキスト3</p>

CSS で class 属性を指定する際は、.を付けて class 名を記述します。

[CSS]

.first {
  font: 20px sans-serif;
  color: blue;
}
.second {
  background-color: orange;
}
.third {
  margin: 10px;
  padding: 30px;
  border: 5px solid #000;
}

◆div タグとは

コンテンツをグループ化するタグです。div タグで
対象の要素を囲む事で、1 つのまとまりとしてグループ化できます。
※ div は 「区分・部分」の意味を持つ division の略称です。


● 属性の指定

div タグで各要素をまとめる際は、id 属性や class 属性を指定して CSS を細かく設定します。

[html]

<div id="all_property">
  <h1>Title</h1>
  <div class="p1_property">
    <p>text1</p>
    <p>text2</p>
    <p>text3</p>
  </div>
  <h2>contents</h2>
  <div class="p2_property">
    <p>text4</p>
    <p>text5</p>
    <p>text6</p>
  </div>
</div>



[CSS]

#all_property {
  color: green;
}
.p1_property {
  font: 20px sans-serif;
  padding: 15px;
}
.p2_property {
  background-color: pink;
  margin: 30px;
}

2. Emmet(HTML)

◆Emmet とは

HTML/CSS 等の入力補完機能を持つエディタ用プラグインです。
CodePen では最初から利用できますが、他のエディタを使う場合はプラグインが必要です。


◆HTML タグ

それでは Emmet を学習していきましょう。Emmet にはいくつかの入力規則があり、
それを覚えると様々なパターンを入力できるようになります。

●id 属性

id は 所定のタグの後に #を追記する事で指定できます。
下記コードの[ ]内を入力して Tab キーを押すと、補完内容が出力されます。

[p#]
<p id=""></p>

タグを指定しない場合は div タグがデフォルトで設定されます。

[#]
<div id=""></div>

また、#の後ろに属性名を続けて記述する事も可能です。

[p#first]
<p id="first"></p>

●class 属性

class は タグの後ろに.を付ける事で指定できます。

[p.]
<p class=""></p>

id 属性と同様に、タグを指定しない場合は div タグがデフォルトで設定されます。

[.]
<div class=""></div>

id と同様に、.の後ろに属性名を続けて記述できます。

[p.first]
<p class="first"></p>

● 属性の中身

タグの後ろに[属性=指定の内容]を追記する事で、属性を指定できます。
(下記の例では、href 属性の後ろに target 属性が追加されています)

[a[target=_blank]] <a href="" target_blank=""></a>

● 入れ子

入れ子にするためには、タグとタグの間に>を入れます。

[ul>li]
<ul>
  <li></li>
</ul>

● 並列の展開

同じ階層にタグを展開する場合は、+をタグの間に追記します。

[h1+p+a]
<h1></h1>
<p></p>
<a href=""></a>

演習 2-3
上記のコードを参照し、各 Emmetを実行しましょう。

● 複製

同じタグをまとめて記述するには、タグの後に*数字を追記します。

[p*3]
<p></p>
<p></p>
<p></p>

● コンテンツの記述

開始タグと終了タグの間にコンテンツを記述する場合は、{}を使います。

[p{text}]
<p>text</p>

● 連番

コンテンツを複製し、更に連番を付けたい場合は$を使います。 

[p{test$}*3]
<p>test1</p>
<p>test2</p>
<p>test3</p>

● 組み合わせ

これらの Emmet は、組み合わせて記述する事が可能です。その際は、
上記のルールを順番に並べる事で実行します。

[main>ul#menu>li*3]
<main>
  <ul id="menu">
    <li></li>
    <li></li>
    <li></li>
  </ul>
</main>

◆ スタイルの優先順位

CSS でセレクタに対してプロパティを設定する際に、有効範囲が重なる事があります。
この時、CSS のプロパティ値には優先順位があり、その順序に従って、HTML 要素は指定されます。
いくつかルールを示すと、

  1. 後から記述されたものが優先される
  2. タイプセレクタよりも class セレクタが優先される (タグ名よりも class 属性が優先)
  3. id 属性と class 属性では id が優先される

より詳細なルールもありますが、ここではタグよりも属性の方が優先(特に id 属性)と覚えておきましょう。

[html]

div>h1{title}+p.sample${text$}*3 p#id01.class02{test}




[CSS]

div {
    c:yellow
    c:red
}

p {
    c:green
}

.sample3 {
    c:blue
}

#id01 {
  c: orange;
}

.class02 {
  c: purple;
}

4.フォームの作成

HTML フォームとは

ユーザが「お問い合わせメール」や「検索ボックス」などへ
必要な項目を入力し、サイトやサーバへデータを送るしくみです。
今回はこのフォームの作り方について学習します。




◆form タグ

それでは、form タグについて学習しましょう。
form タグで以下に示すタグを囲い、入力場所を構成する部品を設置していきます。

  • input タグ
  • textarea タグ
  • label タグ
  • select タグ
  • button タグ

また form タグは action 属性を設定します。
action 属性では、送信ボタンをクリックした後の送信先を指定します。
※ いわゆるサーバー側のプログラムのため、今回は使用しません。



●input タグ

テキストや送信ボタン、チェックボックスなど、フォームの種類を
入力する際に用いるタグです。フォームの種類は、type 属性 の値で
設定する事ができます。以下はその一例です。

  • text
  • number
  • password
  • submit
  • date
  • radio
  • checkbox
  • file
<input type="text" />
<input type="number" />
<input type="password" />
<input type="submit" />
<input type="date" />
<input type="radio" />
<input type="checkbox" />
<input type="button" />
<input type="file" />

また、input タグは閉じタグがありません。各フォームについて
テキストを記載する場合は、後述する label タグを利用します。

●label タグ

フォームを作る際に、テキストボックスやチェックボックス等の横に
文字を設置するタグです。label タグは指定方法が 2 通りあります。

●select タグ

セレクトボックス(メニュー)を作成するタグです。
size 属性で選択肢の表示行数を指定し、
選択肢はoption タグ で作成します。 select タグで option タグをみます。
option タグには value 属性があり、
select タグの name 属性と連携させる事でメニューができます。

<select name="fruits" size="1">
  <option value="fruits">apple</option>
  <option value="fruits">orange</option>
  <option value="fruits">peach</option>
  <option value="fruits">banana</option>
  <option value="fruits">grape</option>
</select>
